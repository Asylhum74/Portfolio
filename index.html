Description du site

Ce site vitrine comporte une page d'acceuil moderne comprenant une description de votre restaurant, une gallerie photo, un accès au menu et un espace dédier si votre enseigne propose des événements.

Il comporte également une page pour le menu comprenant plusieurs exemples de plats ainsi que le menu complet téléchargable. Si vous avez envie de changer une plat de ce menu ou un prix, cette page est entièrement modifiable sans que vous ayez besoin de me contacter (voir partie admin).

Enfin la dernière page est une page de contact, comprenant vos informations, la carte google map de votre restaurant et un formulaire de contact sécurisé (voir partie formulaire).
